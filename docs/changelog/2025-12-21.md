# 2025-12-21

## 변경
- 프롬프트 평가: 모범답변(reference) 옵션 추가로 평가 루브릭을 분기하고, 평가자가 순위 대신 점수만 주도록 변경(동점 허용 랭크는 백엔드에서 산출).
- 평가 결과 UI: DataFrame 제거 후 표/리스트로 재구성, 순위·평균점수·평가자별 점수를 요약 테이블로 표시하고 모델별 근거를 분리(복사용 텍스트/JSON 다운로드 유지).
- UI 그래프: Graphviz 기반으로 채팅/프롬프트 평가 흐름을 탭별 토글로 시각화(모델 생성·평가 8개 노드 표시).
- Swagger: `/api/prompt-eval`에 `reference_answer` 옵션 안내, 평가 요약 포맷 최신화.

## 참고
- 평가·그래프 변경 사항은 Streamlit UI에 바로 반영됨. 프롬프트 평가 시 모범답안 입력이 없으면 기존 루브릭으로 동작.
