# 2025-12-24

## 변경
- Streamlit 기본 모델 선택 로그에서 불필요한 반복 디버그 출력 제거.
- Streamlit UI 렌더/연결 상태 로그를 세션당 1회만 출력하도록 조정.
- chat_compare 로그 노이즈 제거(merge/preview_text 반복 로그 삭제) 및 프롬프트 템플릿 포맷 보강.
- prompt_compare 시스템 프롬프트 템플릿에서 JSON 예시 중괄호 이스케이프 처리 보강.
- prompt_compare 반복 디버그 로그 정리(_llm_factory/_build_model_prompt).
- prompt_compare 실행 로직을 모듈별로 분리(schemas/extractors/clients/evaluator/aggregator).
- 서비스 폴더명 통일: `chat_graph` → `chat_compare`, `prompt_eval` → `prompt_compare`.
- 내부 파일 역할명 통일: `helpers`→`prompts`, `runner`→`workflow`, `eval`→`evaluator`, `llm`→`clients`, `models`→`schemas`, `aggregate`→`aggregator`.
- 프롬프트 파일명/설정 키 통일: `chat_compare_system`, `prompt_compare_*`, `prompt_chat_compare_version`, `prompt_compare_version`.
- 실행 스크립트 통일: `scripts/run_app.py` → `scripts/start_app.py`.
- 프롬프트 평가 UI에서 `{question}` 누락 시 경고/실행 차단 추가.
- OpenAI 채팅 모델 목록을 API 응답 기준으로 갱신하고 평가 모델을 최신 상위 모델로 지정.
- Gemini 채팅 모델 목록을 API 응답 기준으로 갱신.
