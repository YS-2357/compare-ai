# 2025-12-25

## 변경
- OpenAI 채팅 모델 목록을 API 응답 기준으로 갱신하고 평가 모델을 최신 상위 모델로 지정.
- Gemini 채팅 모델 목록을 API 응답 기준으로 갱신.
- Perplexity 모델 목록은 API 응답이 비어 있어 기존 목록을 유지.
- Perplexity 모델 목록을 `sonar`, `sonar-pro`, `sonar-reasoning-pro`, `sonar-deep-research`로 확정.
- 기본 모델(가성비 기준) 및 평가 모델(최근 가성비 기준) 재선정.
- chat_compare 요약 함수 호출명 오류 수정(`_summarize_content` → `summarize_content`).
- 사이드바 모델 선택에 ON/OFF 토글을 추가하고, 활성화 모델만 chat_compare에 전달.
- Cohere 응답 content 파싱 실패 대응을 위해 content 정규화 처리 추가.
- 프롬프트 평가 UI에서 `{question}` 누락 시 경고/실행 차단 추가.
