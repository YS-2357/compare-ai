# 2025-12-24 개발 기록

## 로그 품질
- `app/ui/streamlit_app.py`: 기본 모델 선택 디버그 로그를 제거해 중복 출력 완화.
- `app/ui/streamlit_app.py`: 모델 선택/연결 상태/인증 섹션 로그를 세션당 1회만 출력.
- `app/services/chat_graph/nodes.py`, `app/services/chat_graph/summaries.py`: 반복 디버그 로그 제거.
- `app/services/prompt_eval/runner.py`: 반복 디버그 로그 제거(모델 팩토리/프롬프트 빌더).

## 프롬프트
- `app/services/chat_graph/helpers.py`: 포맷 지시문 중괄호 이스케이프 처리로 템플릿 에러 방지.
- `app/services/prompt_eval/runner.py`: 시스템 프롬프트 템플릿 중괄호 이스케이프 처리로 평가 파싱 오류 방지.

## 구조 정리
- `app/services/prompt_eval/models.py`: 평가 결과 스키마 분리.
- `app/services/prompt_eval/extractors.py`: 출처/메타 추출 유틸 분리.
- `app/services/prompt_eval/llm.py`: 모델 팩토리/기본 프롬프트/평가 모델 선택 분리.
- `app/services/prompt_eval/eval.py`: 평가 프롬프트 생성 및 점수 계산 분리.
- `app/services/prompt_eval/aggregate.py`: 점수 집계 로직 분리.
