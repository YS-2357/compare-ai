# 2025-12-05 개발 로그

## 주요 변경 사항
- FastAPI/LangGraph/Streamlit 전반에 관리자 이메일 기반 우회 로직을 정비하고 HTTP 클라이언트 종료 훅을 도입함
- `/api/ask`가 공급자별 모델 덮어쓰기를 허용하고 LangGraph가 해당 값을 반영하도록 리팩터링함
- Streamlit 사이드바에 LLM 모델 선택기와 관리자 상태 초기화를 추가해 UI에서 바로 모델을 전환할 수 있게 함
- LangGraph 프롬프트를 자연스러운 문단형 응답으로 조정하고 관리자 계정의 멀티턴 제한을 해제함

## 테스트/검증
- 로컬에서 `python main.py`로 FastAPI 실행, `streamlit run app/ui/streamlit_app.py`로 UI 실행 후 관리자 계정 로그인으로 기능 확인
- 사이드바에서 모델을 전환하며 `/api/ask` 요청 페이로드에 `models` 필드가 포함되는지 확인함
