# 2025-12-06 개발 로그

## 주요 변경 사항
- FastAPI/LangGraph/Streamlit 전반에 관리자 이메일 기반 우회 로직을 정비하고 HTTP 클라이언트 종료 훅을 도입
- `/api/ask`가 공급자별 모델 덮어쓰기를 허용하고 LangGraph가 해당 값을 반영하도록 리팩터링
- Streamlit 사이드바에 LLM 모델 선택기와 관리자 상태 초기화를 추가해 UI에서 모델을 쉽게 전환
- LangGraph 프롬프트를 자연스러운 문단형 응답으로 조정하고 관리자 계정의 멀티턴 제한을 해제
- README 및 문서 전반을 단일 리포지터리 배포 흐름과 최신 기능으로 갱신

## 테스트/검증
- 로컬에서 `python main.py`와 `streamlit run app/ui/streamlit_app.py`를 실행해 관리자 계정으로 모델 선택·다중 턴 동작 확인
- `/api/ask` 요청 페이로드에 `models` 필드가 포함되는지, 응답 헤더 `X-Usage-*`가 정상 반영되는지 확인
- Render 배포 명령(백엔드 `uvicorn`, 프론트 `streamlit`)을 문서화하고 환경 변수 분리 설정 검증
